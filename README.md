## 文件介绍

```
.
├── app.js 全局js
├── app.json 全局设置
├── app.wxss 全局样式，公共组件样式封装在这里
├── config.js 域名配置文件
├── images 图片集中营
├── pages 页面集中营
│   ├── index 首页
│   ├── Thearticle 文章内容
│   ├── help 帮助页面 
│   ├── mine 我的 
│   ├── record 兑现记录
│   ├── withdrawal 兑现订单
├── project.config.json
├── wxParse 小程序解析接口传出的html
└── utils 全局封装
    ├── address.js 请求地址配置文件
    ├── request.js 请求封装+环境控制
    ├── time.js 时间转换制作倒计时
    └── util.js 全局方法封装：session_key的获取
```
